# eksamens-projekt-mathilde

Mit eksamensprojekt med udgangspunkt i EXD projektet fra første semester

Jeg har optimeret følgende i mit projekt:
Alle index, css og js filer samt img og audio mapperne

GENERELT:

- korrekt navngivning af filer, img'er og audio
- korrekt mappestruktur - alle css, js, img og audio sammen i semantiske mapper og index filer i root mappen

HTML:

- semantiske tags og classes
- alle png filer er konverteret til svg
- korrekte stier
- ændret sproget på siderne til dansk

CSS:

- animationer
- reset: margin, padding, box-sizing og overflow
- px er erstattet af vh eller vw for et mere responsivt design
- baggrund er nu forskellige elementer i stedet for en samlet png
- responsive design med breakpoints
- alle elementer er blevet erstattet af optimerede versioner med ensrettet design
- glød effekt på skattekisten

JS:

- tooltip og data array slettet
- const variabler tilføjet
- toggle effekter på to fisk, krabben og skattekisterne
- timeout funktioner
- audio objekter
- tilbage button
- click events
- tilføjet dom content loaded funktion
